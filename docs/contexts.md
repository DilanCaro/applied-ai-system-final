# Listening Context Guide

Use this guide when a listener describes an activity instead of naming a genre directly.

- `study`, `focus`, and `coding` requests usually benefit from lower energy, fewer abrupt peaks, and moods like `focused` or `chill`.
- `workout`, `gym`, `running`, and `hype` requests usually align with high energy and moods like `intense`.
- `late-night`, `night drive`, or `after hours` requests can lean `moody` or `chill` depending on whether the user wants momentum or relaxation.
- If the listener wants background audio, favor steadier songs over aggressive peaks.
